# EmDash CMS

A lightweight CMS/blog system styled in grey, black, and white (Neve-inspired), manually deployed on Cloudflare Pages + Workers.

## ✨ Features
- Static site with dynamic JSON-driven pages and blog posts
- Grey/black/white aesthetic theme
- Blog layout with sidebar widgets (categories, recent posts)
- Easy manual deployment without GitHub Actions or terminal
